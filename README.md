## Description

API Rest created to practice what has been studied and learned.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## To See the API Documentation
To view the documentation, start the server ```npm run start:dev ```
Then go to the port where the server is up ( usually localhost:3000 ) /documentation
